Get Started

What is Jewl Shell?

This is a shell that has the speed and good looks of Ghosty or Kitty but with new advanced integrated features. The originals, such as PowerShell, Windows Command Line, and Linux Command Line are very good. However, they all lack modern features since they were designed for old computers. Jewl Shell is built to change that and bring the features that were missing all along. Normally this would require libraries or additional packages, but in Jewl Shell you can just enable and disable these advanced features.

Installation

To install Jewl Shell, follow these steps:

# Step 1: Download the installer
curl -O https://jewl.sh/download/jewl-shell-windows-installer.exe

# Step 2: Run the installer
./jewl-shell-windows-installer.exe

# Step 3 (Optional):
If the program was not run with sudo/administrative permissions then you will need to manually add the bin path.

Features

  • Modern and intuitive interface
  • Advanced command-line features

  • Integrated package management

Configuration

Jewl Shell can be configured using the configuration file located at ~/jewl/config/base.config. Here is an example configuration:

{
    "theme": "dark",
    "installed-modules": ["autocomplete", "syntax-highlighting", "file-driller"],
    "enabled-modules": ["autocomplete", "syntax-highlighting"],
}

Commands

Here are some of the most commonly used commands in Jewl Shell:

                
# List files in the current directory
jewl ls

# Change directory
jewl cd /path/to/directory

# Create a new file
jewl touch filename.txt

# Edit a file
jewl edit filename.txt

# Modify the shell 
#~# command-name

FAQ

Q: How do I update Jewl Shell?

A: You can update Jewl Shell by running the following command:

#~# update
